# Haroun AI

شات ذكي باسم Haroun AI.
واجهة HTML+CSS+JS + باك-إند Python + Flask + HuggingFace.

## تشغيل المشروع
1. ضع ملفات الواجهة في GitHub Pages
2. ضع ملفات الباك-إند على Render / Railway / Fly.io
3. عدّل `script.js` لوضع رابط Render بدلاً من `"https://your-backend-url.onrender.com/chat"`
4. افتح الموقع واستمتع بشات Haroun AI
